📘 Week 3: Computer Fundamentals (For Cybersecurity)

🧩 Part 1: How a Computer Actually Works

🧠 Why This Matters?
Before learning hacking…
you need to understand what you’re actually attacking.
Cybersecurity is NOT about tools.
👉 It’s about understanding systems.
If you skip this → later everything feels confusing.

💻 What is a Computer?
In the simplest way:
👉 A computer is a machine that:
Takes input
Processes it
Gives output
🔄 Real-Life Example (Super Simple)
Let’s say you type:
👉 “youtube.com” in your browser
What happens?
You type (input)
Computer processes it
You see YouTube (output)
Looks simple, right?
But internally… a LOT is happening 👇

⚙️ What Happens Inside?
When you open any app or website:
Your system starts a program
It uses memory (RAM)
The brain (CPU) processes instructions
The operating system manages everything
👉 All these parts work together

🧩 Meet the Main Parts
🧠 CPU (Brain of Computer)
Think of it like your brain
It makes decisions and processes tasks
📌 Example:
When you open Chrome → CPU handles it

🧠 RAM (Short-Term Memory)
Temporary memory
Stores things you are currently using
📌 Example:
Opening multiple tabs = more RAM usage
If RAM is full → system slows down

💾 Storage (Long-Term Memory)
Stores everything permanently
📌 Example:
Photos
Videos
Installed apps
👉 Even after shutdown → data stays

🧮 Operating System (The Manager)
Controls everything
Connects hardware + software
📌 Examples:
Windows
Linux
macOS
👉 Without OS → computer is useless

🔗 Simple Analogy (Easy to Remember)
Think of computer like a human:
CPU → Brain
RAM → Short-term memory
Storage → Long-term memory
OS → Manager controlling everything

⚠️ Why This Matters in Cybersecurity?
Because attackers don’t “hack magic”…
They target these parts:
Running programs
Memory usage
System behavior
Weak system control
👉 If you understand system → you understand attack surface

💡 Key Takeaway
Cybersecurity doesn’t start with hacking…
It starts with understanding.
👉 If you truly understand how a computer works,
you don’t just use tools — you know what’s happening behind them.
And that’s what separates beginners from real professionals.
